## Description
Developed an interactive Shiny dashboard using real-world datasets. Applied regression and exploratory analysis to examine behavioural and seasonal patterns.

- Data visualisation

## Outcome
- Interactive dashboard allows analysis of trends and seasonal effects
- Demonstrates ability to extract insights from large datasets and communicate visually
